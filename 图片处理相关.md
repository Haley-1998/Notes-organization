
## 提取图片的名称（ls，split，extractBefore）
```matlab
% 定义图片文件路径

picpath = 'D:\path\';
pic_name=ls([picpath,'\*.png']);
 
% 示例文件名列表

pic_name= {
    'image_123_part_456.png'
    'image_789_part_012.png'
    'photo_321_part_654.png'};
 
% 初始化pic_cur和pic_num数组

pic_cur = [];
pic_num = [];
 
for i = 1:size(pic_name, 1)
    
    temp = split(pic_name(i, :), '_');% '_'分割文件名
 
    pic_cur{i, 1} = str2double(temp{2});
    pic_num{i, 1} = str2double(extractBefore(temp{4}, '.'));% 提取'.'前面部分
end
 
% 显示提取结果

disp('pic_cur:');
disp(pic_cur);
 
disp('pic_num:');
disp(pic_num);
```



## 根据图片名称复制图片到新文件夹提取数据 copyfile， ismember

```matlab

%根据图片名称复制图片到新文件夹提取数据

%根据tabulate结果，选取0529-06、07

ind1=find(ismember(time2,'20210529-06')==1);
ind2=find(ismember(time2,'20210529-07')==1);

%提取的图片名称

active1=pic_name([ind1;ind2],:);

%原图片所在路径

path='C:\Users\huiyi\OneDrive\Desktop\延硕论小论文\BFB\';

%新图片所在的路径

path2='C:\Users\huiyi\OneDrive\Desktop\延硕论小论文\BFB\0607\';

if ~exist(path2)
    mkdir(path2)
end

%后缀往往有空格，删除后续再重新补充后缀

for i=1:size(active1,1)
    temp=extractBefore(active1(i,:),'.');
    active_name=[temp,'.png'];

    %复制图片

    copyfile([path,active_name],[path2,active_name]);
end



```
